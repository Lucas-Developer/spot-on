2015.05.29
  1. Added libspoton_delete_urls() to libspoton. Version updated to 0.10.
  2. Allow adjustment of the iteration count in the file encryption tool.
  3. Allow configuration of alternating row colors of Chat, E-Mail, and URLs participants. Please see Options.
  4. Allow configuration of the kernel's cache purge timer.
  5. Allow importing of URLs without Import Credentials. Only plaintext URLs will be imported.
  6. Assume zero pointer defaults in Qt::value().
  7. Bound size_t parameter of memcpy() in spoton_crypt::setHashKey(), spoton_misc::peerAddressAndPort(), spoton_sctp_server::setMaxPendingConnections(), spoton_sctp_socket::localAddressAndPort().
  8. Close progress dialogs instead of hiding them. Please see https://en.wikipedia.org/wiki/Shroedinger%27s_Cat.
  9. Compute automatic digests in threads. Safely interrupt on process termination. Completed futures are occasionally removed.
  10. Corrected importing of public keys. Cannot assume that key pairs exist.
  11. Corrected re-encoding of listeners_accounts. Must include the listener_oid in the where-clauses.
  12. Corrected re-encoding of listeners_allowed_ips. Must include the listener_oid in the where-clauses.
  13. Corrected removal of URLs having special characters.
  14. Corrected setting of remote IP information. Please see Neighbors.
  15. Corrected use of spoton::addFriendsKey().
  16. Custom status messages. A maximum of 64 characters will be imposed.
  17. Decreased message size of Rosetta output.
  18. Denote extreme time deltas in Chat. Currently configured to thirty seconds. Source only.
  19. Derive subsequent file encryption key from initially-derived key via SHA-256 (Key_(i + 1) = SHA-256(Key_i)).
  20. Digital signatures should vouch for private key information.
  21. Display URL database connection information within the Search panel.
  22. Do not delete URLs from shared.db unless they are properly loaded.
  24. Do not initiate a rapid StarBeam if the specified participant is temporary.
  25. Dual-purpose domain check boxes. Please see URLs.
  26. Encapsulate data into streams and decrease Base-64 conversions. This will result in smaller messages. Previous functionality will be disrupted.
  27. Establish authentication and encryption keys via silent SMP. The authentication and encryption keys are derived from a Whirlpool digest of the secret.
  28. Guard QList::last().
  29. Guard QList::removeAt().
  30. Many integer conversion corrections.
  31. Modified QFileInfo::lastModified() usage. It's possible that a less-than-or-equal to comparison will leave containers in unpleasant states.
  32. Notify the user of URL import statistics.
  33. Populate URL polarizers before performing an import.
  34. Prompt before accessing a remote URL.
  35. Re-encode URL common and import credential information.
  36. Removed input masks from IPv6 fields.
  37. Replace spaces with dashes in StarBeam and e-mail attachment file names.
  38. Replaced the itemClicked() signal on the e-mail table with itemDoubleClicked() as itemClicked() and itemSelectionChanged() may cause duplicate dialog prompts.
  39. Require account salts to contain at least 512 bytes.
  40. Require the messages in 0002a and 0002b to contain at least 64 bytes.
  41. Simplified copying and exporting of public keys.
  42. The method QTableWidget::findItems() may return items that are irrelevant to the query. Implemented a special column-based method.
  43. Updated NTRU files (05/14/2015).
  44. Verify a StarBeam's provided pulse size against some defined maximum.
2015.04.09
  1. Added Documentation/Spot-On-Kernel.1.
  2. Added the option 65536 to the File Encryption's Read Size.
  3. Allow modification of StarBeam pulse sizes.
  4. Automatically verify StarBeams. Please see the Options panel.
  5. Corrected QSqlQuery preparation. QSqlQuery::exec() must not be used to prepare SQL statements. (spoton_listener::slotNewConnection(), spoton_misc::savePublishedNeighbo())
  6. Corrected Qt DLLs in the Windows bundle.
  7. Corrected inspection of QSqlQuery::exec() errors in various locations.
  8. Corrected server-side processing of UDP datagrams.
  9. Corrected spoton_misc::prepareUrlDistillersDatabase() and spoton_misc::prepareUrlKeysDatabase() with respect to the ok variables.
  10. Improved error messages in the UI.
  11. Include a file's digest within each StarBeam pulse.
  12. Issue QThread::wait() after QThread::quit().
  13. On Qt 5.x and Windows, assume a fusion style.
  14. QWidget::activateWindow() should only be used on Windows!
  15. Updated SQLite to version 3.8.9, Windows.
